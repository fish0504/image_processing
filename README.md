# study_image_processing

研究(卒論)で使用しているプログラムのバックアップとしてのレポジトリです。
作業記録も兼ねています

## システムの概要
画像処理用PCでカメラ入出力から姿勢推定、姿勢推定、ステレオマッチングを行い、AI(Dex-Net)に深度画像を入力し把持位置を計算します。計算した把持位置をリアルタイムコントローラにUDP通信で送信します。

### 実装する機能

1. カメラ入出力
2. 姿勢推定(1自由度)
3. ステレオマッチング
4. AI(Dex-Net)へ入力
5. UDP通信
6. カメラキャリブレーション

### 現在実装が終わった機能

2. ○姿勢推定(1自由度)
